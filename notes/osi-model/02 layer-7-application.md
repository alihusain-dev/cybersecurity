# layer 7 - application layer

## what is the application layer?

- closest layer to the user
- where users interact with network services
- applications use this layer to send and receive data

## examples

- web browser
- email
- messaging apps
- file transfer

## key terms

### gui (graphical user interface)

- the visual interface users interact with
- buttons, menus, windows, and icons

### dns (domain name system)

- translates domain names into ip addresses

example:
- google.com → 142.250.x.x

## analogy

like placing an order at a restaurant.

- you tell the waiter what you want
- everything else happens behind the scenes

the application layer is the part you directly interact with.

## key takeaway

- application layer = user interaction
- closest layer to the user
- users interact with applications, not the lower network layers
